# Salon Banner Studio

## v2.0 Direct Render & Seamless Refresh

- Preview: 750 × 250 px
- Export: 3000 × 1000 px
- Original photos are drawn directly into the export canvas
- No 750px-to-3000px enlargement step
- High-quality image smoothing
- Refreshed seamless crossfade compositor
- No white overlay at image boundaries
- 4 or 5 image layout
- Drag to reorder
- Horizontal and vertical adjustment
- Zoom and horizontal flip
- Sticky preview
- Font, text color, shadow controls
- Text positions: top center, center, bottom center, bottom left, bottom right
- PNG export
- iPhone friendly

The seamless function is a browser-based crossfade, not AI background generation.
